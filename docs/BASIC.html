BASIC

An implementation of TinBASIC is the only HLL language which is ported to R3X VM.

Let's begin with a "Hello, World!"

	REM This is a comment
	REM Print Hello, world to screen
	PRINT "Hello, World!"
	REM Exit.
	END
Using a text editor, save this file as "hello.bas", and run tbc, with the following:
tbc hello.bas > hello.asm
Now it should generate an assembler file, however if you use FASM to compile it, it gives an error:
fasm: 'include/libR3X.pkg' not found.
You need the "libR3X.pkg" and the "r3x_asm.pkg", in an "include" directory, which should be where you generate your progrmas.
Then use:
fasm hello.asm
This should generate a "hello.exe" file, run the program using:
rxvm -exe ./hello.exe
You should see the words "Hello, World!" on the output screen. If you press a key, then the program will exit.

Integers

Unfortunately, as of it's current state, TinyBASIC for R3X doesn't support strings (even C-like strings!). The only
available type is an integer, what's worse is that the only integer names which can be used are A-Z. Hence, there can only be 26 total
integers in all. This limitation is being worked on.
To get started let's know about the "LET" keyword:
REM Set X to 23 / 5
LET X = 20 / 5
REM Y = 4 now
LET Y = (X/2)+5
REM Print Y
PRINT "Y is : "; Y
END
