REX Virtual Machine Documentation

Using the REX VM

Compiling

First install OpenGL, SDL, SDL_ttf, and SDL_image libraries.
Install FASM to compile programs.
Install the GNU C Compiler and glibc. (or libcs should work fine).

Compiling: Option 1

Ensure you system has a bash interpreter. (or anything compatible).
Under "src/build.sh" change ARCHFLAGS to "-m32" if using a 32-bit system,
Run "src/build.sh" with executable permissions (on UNIX-compliant systems use "chmod +x ./build.sh"
Done.

Compiling: Option 2

Ensure you've got GNU Make. (Not tested with BSD Make)
Under "src/Makefile

Running

Run with: ./r3x_vm.out "name of executable"
Press CTRL+C to start debugging during execution. During a segfault, the debugger will automatically pop up,
